## üõ† Installation & Deployment

### Local Network Deployment
1. Clone or download AVA CORE files
2. Install dependencies: `pip install -r requirements.txt`
3. Set OpenAI API key: `export OPENAI_API_KEY=your_key`
4. Run: `python app.py`
5. Share the IP address with your family/team

### Cloud Deployment
```bash
# Deploy to AWS Lambda
python deploy.py aws --region us-east-1

# Deploy to Azure Functions
python deploy.py azure --resource-group ava-core

# Deploy to Google Cloud
python deploy.py gcp --project your-project-id
```

### Docker Deployment
```bash
# Build container
docker build -t ava-core .

# Run on network
docker run -p 5000:5000 --network host ava-core
```

## üö® Troubleshooting

### Can't Connect from Other Devices
1. Check if all devices are on the same network
2. Verify firewall isn't blocking port 5000
3. Try the computer's IP address directly
4. Restart AVA and note the displayed IP

### Voice Not Working
1. Check microphone permissions in browser
2. Ensure speakers/headphones are connected
3. Try the manual text input instead
4. Voice features require local hardware

### AI Responses Not Working
1. Verify OpenAI API key is set correctly
2. Check internet connection for AI features
3. Local conversation mode still works offline
4. Contact support if API key issues persist
